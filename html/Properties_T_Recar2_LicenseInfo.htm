LicenseInfo - свойства

Тип LicenseInfo предоставляет следующие члены.

Свойства
  ИмяОписание
ImageProcessChannelCount
Возвращает количество каналов обработки изображений.
Owner
Возвращает имя владельца ключа защиты и лицензии.
Speed
Возвращает тип канал, показывающего максимальную скорость ТС, при которой оно будет распознаваться.
VideoOnlyViewChannelCount
Возвращает количество каналов просмотра видео.
VideoProcessChannelCount
Возвращает и устанавливает количество каналов обработки видео.
